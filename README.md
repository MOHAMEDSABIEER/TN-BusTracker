# 🚌 TN Bus Tracker

A real-time bus tracking application designed for Tamil Nadu. This app helps users find bus routes, check live schedules, and plan their commute across the state.

## ✨ Features
* **Live Tracking:** Real-time updates on bus positions and ETAs.
* **Route Discovery:** Search for buses by route number or destination.
* **Responsive Design:** Optimized for both mobile and desktop users.
* **Modern UI:** Clean interface built with React and Tailwind CSS.

## 🛠️ Tech Stack
* **Frontend:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Deployment:** [Vercel](https://vercel.com/)
